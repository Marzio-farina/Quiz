# Come pulire la cache di electron-updater

## Windows

La cache di electron-updater si trova in:
`%LOCALAPPDATA%\electron-updater\cache\`

Per pulirla:
1. Chiudi completamente l'applicazione
2. Apri Esplora File
3. Incolla nella barra degli indirizzi: `%LOCALAPPDATA%\electron-updater\cache\`
4. Elimina tutti i file e cartelle in questa directory
5. Riavvia l'applicazione

## Alternativa rapida

1. Chiudi l'applicazione
2. Premi `Win + R`
3. Digita: `%LOCALAPPDATA%\electron-updater\cache\`
4. Elimina tutto il contenuto
5. Riavvia l'app
